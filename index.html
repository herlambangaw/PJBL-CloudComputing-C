Cloud Computing dengan AWS

Cloud computing adalah teknologi yang menggabungkan penggunaan komputer dan internet untuk menyajikan berbagai layanan melalui internet. Dalam pengertian yang lebih luas, cloud computing meliputi berbagai sumber daya seperti aplikasi, penyimpanan data, server, database, jaringan, dan perangkat lunak yang dapat diakses secara online. Dalam pengoperasian cloud computing, terdapat lima komponen yang harus ada, yaitu perangkat keras, jaringan perangkat, penyimpanan, server, dan alat virtualisasi. Perangkat keras adalah dasar infrastruktur cloud yang menyimpan kode dan fungsi pada sistem yang akan dibangun. Jaringan perangkat berfungsi sebagai alat transmisi dan komunikasi untuk mengoordinasikan beberapa perangkat yang terhubung. Penyimpanan memungkinkan pengguna untuk menyimpan file selama mereka memiliki akses ke internet. Server adalah bagian dari infrastruktur cloud yang berfungsi sebagai pusat pengolahan data. Alat virtualisasi memungkinkan beberapa sistem operasi untuk berjalan pada satu perangkat keras, sehingga memungkinkan penggunaan sumber daya yang lebih efisien.

Ada tiga jenis layanan cloud computing yang tersedia, yaitu Software as a Service (SaaS), Platform as a Service (PaaS), dan Infrastructure as a Service (IaaS). SaaS menyajikan aplikasi melalui internet, seperti Adobe Creative Cloud yang memungkinkan pengguna mengakses aplikasi melalui model subscription. PaaS menyajikan platform untuk pengembangan aplikasi, seperti Google App Engine yang memungkinkan pengembang membuat aplikasi tanpa perlu membangun infrastruktur sendiri. IaaS menyajikan infrastruktur, seperti server, penyimpanan, dan jaringan, yang dapat diakses secara online, seperti Amazon Web Services (AWS) yang memungkinkan pengguna membuat dan mengelola infrastruktur sendiri.

Kelebihan cloud computing meliputi irit biaya, meningkatkan produktivitas, kecepatan, efisiensi, performa, dan keamanan. Kekurangan cloud computing meliputi ketergantungan pada internet, biaya tambahan untuk karyawan tambahan, risiko keamanan, batasan bandwidth, dan batasan storage. Contoh pemanfaatan cloud computing meliputi penyimpanan data, backup dan pengambilan data, membuat dan mengetes aplikasi, menganalisis data, dan email layanan seperti Zimbra Mail. Beberapa penyedia layanan cloud computing yang populer di Indonesia adalah Indonesian Cloud, yang menyediakan layanan cloud server, private cloud, dan VPS.

HTML

HTML adalah kependekan dari Hypertext Markup Language, yang merupakan sebuah bahasa markup yang digunakan untuk membuat struktur halaman suatu website yang menarik, saling terhubung satu dengan yang lainnya, dan yang pasti dapat diakses melalui internet. HTML ditemukan oleh Tim Berners-Lee pada tahun 1991 dan awalnya digunakan untuk membantu ilmuwan dalam mengakses dokumen, namun kini HTML semakin berkembang pesat di dunia pemrograman web.

CSS

CSS adalah singkatan dari Cascading Style Sheets, yang digunakan untuk menentukan gaya dari tampilan website yang akan dibuat. CSS memungkinkan pengembang untuk mengatur tampilan halaman website dengan cara yang lebih efektif dan efisien, serta memudahkan dalam mengubah atau mengedit tampilan website. CSS pertama kali diusulkan oleh Håkon Wium Lie pada 10 Oktober 1994, dan pada tahun 1996, World Wide Web Consortium (W3C) menghasilkan Rekomendasi W3C CSS (CSS1) pertama. lebih cepat.

JavaScript

JavaScript adalah sebuah bahasa pemrograman yang digunakan untuk membuat halaman web yang interaktif. Dengan JavaScript, pengembang dapat membuat fungsi-fungsi yang terkait dengan berbagai peristiwa, seperti klik mouse atau kursor mouse, untuk membuat perubahan pada HTML dan CSS. Alat pengembang JavaScript meliputi debugger, analisis program statis, profiler, dan editor teks yang mendukung sintaks JavaScript.
